Detailed Description

Represents a renderable 3D geometric object.

The Mesh class holds the vertex data (position, normals, texture coordinates) and face indices that define the geometry. It inherits from Eng::Node to be part of the scene graph and is associated with an Eng::Material for rendering properties.

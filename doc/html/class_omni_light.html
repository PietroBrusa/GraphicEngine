Detailed Description

Implements an omnidirectional (point) light source.

An OmniLight emits light equally in all directions from a specific point in space. Its intensity typically diminishes over distance based on the attenuation factors inherited from the Eng::Light base class. This is used to simulate light bulbs and small local sources.

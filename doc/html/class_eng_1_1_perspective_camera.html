Detailed Description

Implements a camera with a perspective projection.

The PerspectiveCamera simulates how objects appear smaller as their distance from the viewer increases. It is defined by its Field of View (FOV) and Aspect Ratio, which create a viewing frustum that is a truncated pyramid.

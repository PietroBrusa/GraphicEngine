Detailed Description

Abstract base class for all light sources in the scene.

The Light class inherits from Eng::Node, allowing it to be placed and oriented within the scene graph. It holds the fundamental lighting properties: color components (ambient, diffuse, specular), attenuation factors, and a unique OpenGL-style ID. Concrete types (like OmniLight, SpotLight) derive from this class.

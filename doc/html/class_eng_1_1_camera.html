Detailed Description

Abstract base class for all cameras in the engine.

The Camera class inherits from Eng::Node, allowing it to be positioned and oriented within the scene graph. It manages the projection matrix and defines the near and far clipping planes. Concrete camera types (e.g., Perspective, Orthographic) will derive from this class and implement their specific projection logic.

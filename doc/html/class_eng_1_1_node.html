Detailed Description

Represents a single element in the hierarchical Scene Graph.

The Node class is the basic building block for organizing a 3D scene. Each node holds a local transformation matrix and manages a list of child nodes. The final world transformation is computed by concatenating the node's local matrix with all its parent nodes' matrices.
